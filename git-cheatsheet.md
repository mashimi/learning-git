+ ### Git Branches

      1.  git branch -a
      2.  git branch blahblah
      3.  git log
      4.  git log --oneline
      5.  git checkout blahbranch
      6.  git branch -m blah1 blah2
      7.  git branch -d blah2
      8.  git checkout -b blahblah
      9.  git commit -am "blah"
      10. git diff master blahbranch
      11. git merge blahbranch
